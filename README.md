## 2. 程式設計與資料結構 (TKU_114-3_DS)

本專案使用日期作為資料夾名稱來管理每堂課的作業與實作程式碼。

### 7/2 實作與作業內容

包含 Java 開發環境建置、基本語法、變數、資料型態、鍵盤輸入（Scanner）與基本運算。

#### 課堂練習與作業列表

1. **[HelloJava.java](./0702/HelloJava.java)**：基礎 Java 測試，輸出個人化歡迎詞。
2. **[ExecutionOrder.java](./0702/ExecutionOrder.java)**：示範 Java 程式碼的順序執行流程。
3. **[DailyRoutine.java](./0702/DailyRoutine.java)**：輸出個人的早上日常流程三步驟。
4. **[ProfileCard.java](./0702/ProfileCard.java)**：輸出個人名片，並包含程式功能與欄位說明註解。
5. **[UserInfo.java](./0702/UserInfo.java)**：宣告變數儲存姓名、學號、科系與期望成績並輸出。
6. **[CourseInfo.java](./0702/CourseInfo.java)**：宣告變數儲存課程名稱、學分、時數與必選修屬性並輸出。
7. **[MyScoreAverage.java](./0702/MyScoreAverage.java)**：計算三科固定成績（Java、English、Math）之平均。
8. **[GreetingInput.java](./0702/GreetingInput.java)**：使用 `Scanner` 讀取姓名與年齡，輸出問候語。
9. **[HealthCalculator.java](./0702/HealthCalculator.java)**：**個人健康資料計算器**（課堂整合實作），輸入身高體重並計算 BMI。
10. **[ScoreReport.java](./0702/ScoreReport.java)**：**成績資料計算器**（課後作業），輸入三科成績計算小數平均分數，包含詳細註解。
